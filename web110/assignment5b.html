I created an external style sheet titled "mystyle.css" and applied it to the pages "Assignment 1" through "Assignment 5a." Here is a link to the index page to view the changes: Index Page

This page is an example of internal styling, where I set the background color to MintCream, the Heading 1 color to DarkOliveGreen, and the paragraph color to Olive. I also used the Courier New font family and set the font size of Heading 1 to 125% and the paragraph text to 100%.

This is an example of inline styling, where I changed the paragraph color to maroon and the font family to Georgia.

Through this lesson, I learned how external, internal, and inline CSS styles can be used to control the appearance and consistency of web pages. I gained a better understanding of how external style sheets allow multiple pages to share the same design, while internal and inline styles provide flexibility for page-specific or element-specific formatting. I also learned how font families, colors, background colors, and text sizing can improve readability and user experience. These concepts can be applied in accounting, marketing, education, and general business environments to create professional, organized, and visually appealing websites that effectively communicate information to users. In accounting, these concepts can be used to create professional web pages that clearly present financial information and reports. In marketing, CSS can help build visually appealing websites that strengthen branding and engage customers. In general business, consistent web design can improve communication, enhance professionalism, and make information easier for employees and clients to access.
